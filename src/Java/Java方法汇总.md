# Java集合



## Collection

1. **boolean add(E e)**

   > **添加一个元素**


2. **boolean addAll(Collection<? extends E> c)**

   > **将集合中的所有元素添加到其他集合中**


3. **clear()**

   > **暴力清除集合中所有元素**

4. **boolean contains(Object o)**

   > **判断集合是否包含某个元素**，包含返回true

5. **boolean isEmpty()**

   > **如果此集合不包含元素**，则返回true

6. **int size()**

   > **返回集合个数**
