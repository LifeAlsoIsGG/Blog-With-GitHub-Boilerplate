将两个整数数组合并之后按照升序排序,并打印输出到控制台,注意不要去重

public class 合并数组按照升序排序 {
    public static void main(String[] args) {
        int a[] = {1,3,5,7};
        int b[] = {2,4,5,8};
        int c[] = new int[a.length + b.length];
        int i =0, j = 0 , k = 0;
        while(i < a.length && j < b.length){
            if(a[i] < b[j]) {
                c[k++] = a[i++];
            }else{
                c[k++] = b[j++];
            }
        }
        while (i < a.length) {
            c[k++] = a[i++];
        }
        while (j < b.length) {
            c[k++] = b[j++];
        }
        System.out.println(Arrays.toString(c));
    }
}

给定一个数组，设计一个既高效又公平的方法随机打乱这个数组。

Leetcode

剑指 Offer 03. 数组中重复的数字

  • https://leetcode-cn.com/problems/shu-zu-zhong-zhong-fu-de-shu-zi-lcof/

第一种：使用HastSet去重，遍历添加时如果返回为false则说明重复则直接返回此数

public static int findRepeatNumber() {
        HashSet<Integer> dic = new HashSet<Integer>();
        int[] nums = {1,2,2,4};
        int repeat = 0;
        for(int num : nums){
            if( !dic.add(num)){
                repeat = num;
                break;
            }
        }
        return repeat;
    }
