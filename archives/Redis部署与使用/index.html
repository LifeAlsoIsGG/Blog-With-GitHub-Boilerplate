flushdb

键Key

参考

  • https://www.runoob.com/redis/redis-keys.html

查看所有key

keys *

设置过期时间

expire [key] [second]

查看剩余时间

ttl [key]

两种情况

  • -1：未设置过期时间
  • -2：已过期或不存在key

查看key对应的value类型

type key

给某个key的value拼接

append [key] [value]

获取key对应value的长度

strlen [key]

根据range截取

#截取字符串[0,3]
getrange [key] 0 3

替换

#从0开始替换为value
setrange [key] 0 [value]

对于value为integer类型的操作

自增长++

incr [key]

自减小--

decr [key]

指定增加大小

incrby [key] [number]

指定减小大小

decrby [key] [number]
